Tweet list

This question provides candidates with an opportunity demonstrate their skills and experience building web applications. The exercise allows candidates to apply their knowledge of front-end technologies and demonstrate their approach to solving problems. Candidates can choose between two flavors of this exercise to suit their experience: React or Vanilla (i.e. no JS framework).

The Problem

Build a twitter clone (simplified).

Features & Requirements

Users of this app should be able to:

  • Add a tweet to a list (using canned ipsem lorem text)
  • Remove the last tweet from the list
  • (Optional) expand tweets to show tweets longer than 160 chars. This feature doesn't require a significantly different implementation approach from the above features, so can be skipped if time is short.
The app should:
  • only present the remove button when there are tweets to remove
