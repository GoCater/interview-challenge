This challenge provides candidates with an opportunity demonstrate their skills and experience building web applications. The exercise allows candidates to apply their knowledge of front-end and back-end technologies and demonstrate their approach to solving problems.

Projects

Backend

Basic NodeJS application created with express-generator and using MongoDB as database.

How to run it

  • Install: yarn install / npm install
  • Start MongoDB: mongod --dbpath ~/data/db/ (or any other location)
  • Init the DB: yarn run init / npm run init
  • Run the application: yarn start / npm start

Utils scripts

In order to populate the DB with some tweets you can run: yarn run create-tweets / npm run create-tweets.
The script itself is located at backend/scripts/create-tweets.js and you can also run it the parameter h to see the avilable options.

Frontend

ReactJS application created with create-react-app. It includes Hot Reloading so there is no need to re-start the server after any change.
