Tweets list

Features & Requirements

Users of this app should be able to:

  • Load tweets by clicking the button Load more, this could be done by either consuming an API endpoint or using the tweet JSON data.
  • Clear the list of tweets and be able to load them since the beginning again.
  • Switch between Tweets List and Tweet creation form by clicking the navbar
  • Create Tweets from the creation form
The app should:
  • Load 2 tweets when user lands on the page, load more in batches of 2 per click
  • Only display the Load more button when there is more data to load
  • Only display the Clear list button when there are tweets to remove
Checkout the demo!

Optional features:
  • Add validation to the Tweet creation form, refer to the validation rules file
  • Only display 160 chars of the tweet content and add a link to expand it (see screenshot)
